A modern, responsive **Digital Identity Card Website** built using HTML, TailwindCSS, and vanilla JavaScript.
This project showcases a personal profile with interactive UI, smooth transitions, and a clean glassmorphism design.

📖 Overview

This project represents a **digital personal identity platform** designed for members of HMPS PGMI UIN Malang.
It combines aesthetic UI design with lightweight performance, making it ideal for:

* Personal branding
* Student organization identity
* Portfolio introduction page

✨ Key Features

* 🧊 **Glassmorphism UI Design**
* 🎯 **3D Button & Card Effects**
* 🌙 **Dark Mode Toggle**
* 🔄 **Single Page Navigation (SPA-like behavior)**
* 🎨 **Animated Background (Liquid Blob Effect)**
* 📱 **Fully Responsive (Mobile-first)**
* 🔗 **Integrated Social Media Links**
* ⚡ **Lightweight & Fast (No Frameworks)**

🧭 Usage Guide

* Click **"Profil Identitas Diri"** to navigate to detailed profile page
* Use the **right-side smart panel** to:

  * Toggle Dark Mode
  * Access external links
* Use browser **back button** to return to home

🎨 UI Highlights

* Clean card-based layout
* Smooth animation transitions
* Floating smart panel (settings)
* Instagram-style profile ring
* Glass + blur effects

📌 Customization

You can easily customize:

* 👤 Profile image → change image URL
* 📝 Personal data → edit HTML content
* 🎨 Theme colors → modify CSS variables
* 🔗 Social links → update anchor tags
